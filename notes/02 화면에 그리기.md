## 01 창 만들기

- 프로그램: 자신의 창을 제어
- 데스크톱 환경: 화면을 제어
- `tkinter`: `Tk`라고 불리는 그래픽 툴킷 지원

```python
import tkinter

window = tkinter.Tk()
tkinter.mainloop()
```

```python
# tkinter.mainloop()을 호출하면 다음의 ✨이벤트 루프가 실행

while True:
  for evt in pendingEvents():
    handleEvent(evt)
  drawScreen()
```

- `pendingEvent()`: 데스크톱 환경으로부터 마우스 클릭이나 키 입력 등 쌓여 있는 최근 이벤트를 얻음
- `handleEvent()`: 애플리케이션의 상태를 업데이트
- `drawScreen()`: 창을 다시 그림
  <br/>

## 02 창에 그리기

```python
import tkinter

WIDTH, HEIGHT = 800, 600
window = tkinter.Tk() # 창 생성
# 생성한 창에 대한 Canvas 생성
canvas = tkinter.Canvas(window, width=WIDTH, height=HEIGHT)
canvas.pack()
```

## 04 텍스트 스크롤하기

- 일반적으로 브라우저는 페이지 좌표계로 페이지를 **레이아웃**하고 난 뒤에 화면 좌표계로 페이지를 렌더링
- **디스플레이 리스트 (display list)**: 화면에 그려야 할 요소의 집합
  <br/>

## 05 더 빠른 렌더링

- 브라우저와 같은 그래픽 애플리케이션은 보통 **프레임 레이트** (frame rate; 화면 갱신 속도)를 정확히 60Hz 이내에 다시 그리는 것을 목표로 함 (모든 작업을 60분의 1초 이내로 완료해야 함)
- ➡️ 16ms를 애플리케이션의 **애니메이션 프레임 버짓 (animation frame budget)이라고 함**
